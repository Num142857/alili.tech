Raspberry Pi doesn't come with hard drive, so if you want to install system, need to prepare TF card or USB drive yourself.

Next, need to go to Raspberry Pi official website to download system.
Official installation system tool, NOOBS is very simple.

[Download Address](https://www.raspberrypi.org/downloads/noobs/)

After download completes, we can install system according to following steps.

1, Extract downloaded zip package to TF card

![](http://www.geekfan.net/wp-content/uploads/img_5249b9158d98d.png)

2, Insert TF card into Raspberry Pi, and connect HDMI to monitor, mouse keyboard, connect power.

At this time, Raspberry Pi lights up, monitor will display following content:


![](http://www.geekfan.net/wp-content/uploads/img_524d91f199b4f.png)

3, At this time press shift key, will enter recovery system interface

![](http://www.geekfan.net/wp-content/uploads/img_524d9fb0d8009.png)

Select system you want to install, click install.

Wait a few minutes, system is installed.

![](http://cms-bucket.nosdn.127.net/catchpic/7/7b/7b1fc28118e3151df0ef1dca46241171.jpg?imageView&thumbnail=550x0)

After restarting into desktop, this desktop operating system is called PIXEL OS; Based on Debian.

PIXEL means "Raspberry-Pi Improved Xwindows Environment, Lightweight", meaning "Raspberry Pi improved lightweight Xwindows environment".
