## Enable Forced Dark Mode

### Chrome
Enter browser, enter in address bar:

```bash
chrome://flags/#enable-force-dark
```

### Edge

Enter browser, enter in address bar:

```bash
edge://flags/#enable-force-dark
```

### Operation

Change first item Disabled to Enabled, then restart browser
