2020-02-02
0
Game

4NXCI 是将 XCI 文件转换为 NSP 的工具

目前只支持windows,其他系统不支持

Github地址:

https://github.com/The-4n/4NXCI

下载地址

https://github.com/The-4n/4NXCI/releases

使用方法

GUI工具

命令用不习惯的可以下载GUI的版本,文件命名大致为4nxci-v4.03_GUI.zip

命令行工具

.\4nxci.exe [options...] <path_to_file.xci>  
  
Options:  
-k, --keyset             设置keyset文件路径, 默认为 ./keys.dat  
-h, --help               显示帮助信息
-t, --tempdir            设置临时文件夹
-o, --outdir             设置导出文件夹
-r, --rename             设置导出的nsp文件的文件名
--keepncaid              保持当前ncas ID
